# 🧮 Matrix Operations Tool

A Python-based tool for performing common matrix operations using NumPy.

---

## 🚀 Features

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose
- Matrix Determinant
- Matrix Inverse

---

## 🛠️ Tech Stack

- Python
- NumPy

---

## ▶️ Run

```bash
pip install -r requirements.txt
python matrix_operations.py
```
